Based on the <a href="https://github.com/dylanaraps/torque">original project</a> from <a href="https://github.com/dylanaraps">dylanaraps</a>

A simple TUI client for `transmission-daemon`.

## Dependencies

- `bash`
- `transmission-cli`

## Running

1. Start `transmission-daemon`.
2. Run `tormix`.
